Ajoutes un rôle à un utilisateur.

Paramètres
COOKIEPHPSESSID : le cookie de session POST utilisateur_id : l'id de l'utilisateur POST role_id : l'id du role
Renvoie
- code reponse: 200 : le rôle a été attribué avec succès 400 : erreur requête 401 : non autorisé 409 : le role n'a pas pu être attributé (role déjà attribué) 503 : la connection à la base de données a échoué
